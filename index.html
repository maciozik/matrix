How to play

The purpose of each round of the game is to reveal the 3 digits randomly activated among the 9 digits of the grid.

You can selected 3 different digits per attempt. The number of right digits found among the 3 digits you selected is displayed at the end of each attempt (*), but you will not know which of the 3 digits selected are right and which ones are wrong.

Once the 3 right digits revealed, 3 new digits are randomly activated and a new round begins. You can make as many attempts as you want per round, and as many rounds as you can within the time limit.

You can select digits by touching/clicking on the cases or by using the digits from 1 to 9 on your numpad. Run a new game by selecting a first digit.

How to earn points

You can increase your score after each round you win. The number of points earned is determined by the time you spent and by the number of attempts you make to complete the round.

But be careful: you lose points each time you select 3 digits that you already selected earlier in the round, and by spending too much time between each attempt.

How can I report a bug

You can report bugs directly on the GitHub repository of the game.
