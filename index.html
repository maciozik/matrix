How to play

The purpose of the game is to uncover the 3 right boxes randomly activated among the 9 boxes of the grid.

You have to select 3 different boxes per attempt. The number of right boxes found among the 3 boxes you selected is displayed at the end of each attempt (*), but you will not know which of the 3 boxes selected are right and which ones are wrong.

Once the 3 right boxes are found, 3 new boxes are randomly activated and a new round begins. You can make as many attempts as you want per round in order to find the 3 right boxes, and as many rounds as you can within the time limit.

Select a new box directly via the screen, or use the keys from 1 to 9 on your numpad. Run a new game by selecting a first box, or by pressing Enter or Space. Press Escape or reload the page to run a new game.

How to earn points

You can increase your score after each round you win. The number of points earned is determined by the time you spent to complete the round and by the number of attempts you made.

But be careful: you lose points each time you select the 3 same boxes several times on the same round, or by spending too much time between each attempt.

How can I report a bug?

You can report a bug directly on the GitHub repository of the game.

Comment jouer ?

Le but du jeu est de découvrir les 3 bonnes cases activées aléatoirement parmi les 9 cases de la grille.

Vous devez sélectionner 3 cases différentes par essai. Le nombre de bonnes cases trouvées parmi les 3 cases sélectionnées est affiché à la fin de chaque essai (*), mais rien ne vous indiquera lesquelles des 3 cases sélectionnées sont bonnes et lesquelles ne le sont pas.

Une fois les 3 bonnes cases trouvées, 3 nouvelles cases sont activées aléatoirement et un nouveau tour commence. Vous pouvez faire autant d'essais que vous le souhaitez à chaque tour afin de trouver les 3 bonnes cases, ainsi qu'autant de tours que vous le pouvez dans la limite du temps imparti.

Sélectionnez une nouvelle case directement depuis l'écran, ou utilisez les touches de 1 à 9 de votre clavier numérique. Lancez une nouvelle partie en sélectionnant une première case, ou en appuyant sur Entrer ou Espace. Appuyez sur Échap. ou rechargez la page pour lancer une nouvelle partie.

Comment gagner des points ?

You pouvez augmenter votre score après chaque tour remporté. Le nombre de points gagnés est déterminé par le temps que vous avez mis pour remporter le tour, ainsi que par le nombre d'essais effectués.

Mais attention : vous perdez des points à chaque fois que vous sélectionnez les 3 mêmes cases à plusieurs reprises dans le même tour, ainsi qu'en attendant trop longtemps entre chaque essai.

Comment signaler un bug ?

Vous pouvez signaler un bug directement sur le dépôt GitHub du jeu.
